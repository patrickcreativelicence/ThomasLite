### Typography

Many simple text formatting options are ready-made to be added to your templates.

#### Alignment

```css
.text-left
.text-right
.text-center
.text-justify
```

#### Wrapping

```css
.text-nowrap    /* Don't wrap text to new lines */
.text-pre       /* Don't collapse white-space */
.text-ellipsis  /* Truncate long text with an ellipsis... */
```

#### Weight

```css
.text-bold
.text-normal
.text-light
```

#### Size

```css
.text-xxl
.text-xl
.text-lg
.text-md
.text-sm
```

#### Transformation

```css
.text-lowercase
.text-uppercase
.text-capitalize
```
